#### Quick Start
```bash
docker run -d -p 9292:9292 -e FLUENTD_UI_PORT=9292 git.yoursite.com:5005/docker/fluentd:latest
```

#### Note
* The fluentd-ui initial account is admin:changeme, you can change it in dashboad
